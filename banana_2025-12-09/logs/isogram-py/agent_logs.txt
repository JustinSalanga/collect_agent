    """
    Determine if a word or phrase is an isogram.
    
    An isogram is a word or phrase without a repeating letter, 
    however spaces and hyphens are allowed to appear multiple times.

        string (str): The word or phrase to check
        
    Returns:
        bool: True if the string is an isogram, False otherwise
        
    Examples:
        >>> is_isogram("lumberjacks")
        True
        >>> is_isogram("background")
        True
        >>> is_isogram("downstream")
        True
        >>> is_isogram("six-year-old")
        True
        >>> is_isogram("isograms")
        False
    """
    # Edge Case: Empty string is considered an isogram
    if not string:
        return True
    
    # Track seen letters (case-insensitive)
    seen_letters = set()
    
    for char in string:
        # Edge Case: Skip spaces and hyphens as they're allowed to repeat
        if char in (' ', '-'):
            continue
            
        # Edge Case: Non-alphabetic characters are ignored for isogram check

        char_lower = char.lower()
        
        # Edge Case: If we've seen this letter before, it's not an isogram
        if char_lower in seen_letters:
            return False
            
        seen_letters.add(char_lower)
    
    return True

## 2. Common Misinterpretations
### **"Allowed to Repeat" vs. "Ignored" Distinction**
- **The Misinterpretation**: Developers may think spaces/hyphens are *checked* but exempt from the no-repeat rule, rather than being *completely ignored* in the duplicate check.
- **Critical Difference**: 
  - **Correct**: Filter out spaces/hyphens first, then check remaining letters for duplicates
  - **Incorrect**: Check all characters, but allow spaces/hyphens to have duplicates while other characters cannot
- **Why It Matters**: The incorrect approach would fail on "a-a" (hyphen repeats, but letters don't) vs. "aa" (letter repeats). Both should be isograms, but the logic differs.
### **Partial Matching on Substrings**
- **The Misinterpretation**: Using `string.count(char) > 1` without filtering first will count spaces/hyphens and cause false negatives.
- **Example**: `is_isogram("six-year-old")` - counting ' ' or '-' would show multiple occurrences, incorrectly returning `False`.
### **Over-Reliance on Examples**
- **The Misinterpretation**: Assuming from the examples that input will always be lowercase and contain only simple ASCII.
- **Risk**: Fails on "Six-Year-Old" (mixed case) or "thirty‑one" (Unicode hyphen U+2011 vs ASCII hyphen U+002D).
